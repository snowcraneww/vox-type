# VoxType 调研工作区

这个目录保存 VoxType 调研过程中的中间材料。

- `repos/`：第三方仓库本地克隆，仅用于源码阅读。该目录被 Git 忽略，避免把外部项目 vendoring 到本仓库。
- `requirements-brief.md`：对原始需求的结构化理解。
- `repo-candidates.md`：候选仓库清单和筛选记录。
- `solution-analysis.md`：方案对比和推荐结论。

规则：

- 不复制第三方源码到 VoxType。
- 记录来源 URL 和许可证。
- 区分事实、推断和建议。
- 优先链接本地路径和上游 URL，不粘贴大段外部源码。
- 除仓库名、API 名、函数名、命令和专有名词外，调研结论默认用中文写。

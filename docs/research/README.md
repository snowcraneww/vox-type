# Research

This directory stores durable research notes for VoxType.

Rules:

- Record source URLs for every important claim.
- Separate facts, inferences, and open questions.
- Do not copy source code from researched projects.
- Include license notes before recommending reuse.
- Prefer official docs, source repositories, and actively maintained projects.

Planned documents:

- `requirements-brief.md`: structured product requirements after reading the original draft.
- `open-source-landscape.md`: GitHub and ecosystem research for speech input and local transcription.
- `technical-options.md`: comparison of desktop shell, ASR, audio capture, and text injection options.

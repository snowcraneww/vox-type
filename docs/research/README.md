# 调研文档

本目录保存 VoxType 的长期调研文档。面向维护者的调研结论默认使用中文；函数名、API 名、命令、仓库名和专有名词保持原文。

规则：

- 重要结论要记录来源 URL。
- 区分事实、推断、建议和待确认问题。
- 不复制调研项目的源码。
- 推荐复用前必须记录许可证风险。
- 优先参考官方文档、源仓库和仍在维护的项目。

当前文档：

- `requirements-brief.md`：根据原始草稿整理的产品需求。
- `open-source-landscape.md`：语音输入、本地转写和相关开源项目调研。
- `technical-options.md`：桌面壳、ASR、录音和文本上屏技术选项对比。
- `mvp-technical-proposal.md`：Rust + Tauri 2 + React/TS MVP 技术方案草案，等待维护者确认。

🎙️ VoxType: Rust + Tauri + Whisper.cpp 语音输入法

https://img.shields.io/badge/Rust-000000?style=flat&logo=rust](https://www.rust-lang.org/)
https://img.shields.io/badge/Tauri-24C8D8?style=flat&logo=tauri](https://tauri.app/)
https://img.shields.io/badge/License-MIT-yellow.svg](https://opensource.org/licenses/MIT)

VoxType 是一款基于 Rust + Tauri + React 构建的 Windows 桌面语音输入法。  
它利用 whisper.cpp 在本地运行 Whisper 模型，无需联网，无需 Python 环境，即可实现“按住说话，松开上屏”的高效输入体验。

核心理念：离线优先、极致轻量、隐私安全。

💻 核心代码结构

src-tauri/src/lib.rs (Rust 核心)

use std::path::PathBuf;
use std::process::Command;
use tauri::{Manager};

// 停止录音并调用 Whisper 识别
#[tauri::command]
async fn stop_and_transcribe(app: tauri::AppHandle) -> Result<String, String> {
    let exe_dir = app.path().resource_dir().unwrap();
    let whisper_bin = exe_dir.join("bin").join("whisper-cli.exe");
    let model_path = exe_dir.join("models").join("ggml-base.bin");
    let wav_path = exe_dir.join("temp.wav");

    let output = Command::new(whisper_bin)
        .args([
            "-m", model_path.to_str().unwrap(),
            "-f", wav_path.to_str().unwrap(),
            "-l", "zh",
            "-nt", // No timestamps
        ])
        .output()
        .map_err(|e| e.to_string())?;

    Ok(String::from_utf8_lossy(&output.stdout).trim().to_string())
}

#[cfg_attr(mobile, tauri::mobile_entry_point)]
pub fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_global_shortcut::Builder::new().build())
        .invoke_handler(tauri::generate_handler![
            stop_and_transcribe
        ])
        .run(tauri::generate_context!())
        .expect("error on run");
}
